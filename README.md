Steps:

1. clone the project
2. enter to the project folder
3. run npm i
4. run npm run build
5. run npm run serve
6. enter http://localhost:3000/
7. in order to run test, run npm run test in the terminal.

--please note: the convert api which i used is limited for 250 calls per month
